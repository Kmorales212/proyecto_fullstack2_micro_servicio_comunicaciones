\# Proyecto Colegio - Microservicio Comunicaciones



Microservicio encargado de la gestión de Avisos



\## Stack Tecnológico

\- Java

\- Spring Boot

\- JPA / Hibernate



\## Descripción

Este servicio gestiona de forma autónoma su lógica de negocio y persistencia de datos, exponiendo endpoints RESTful para ser consumidos por el BFF.
